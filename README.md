# findcode

  Experimental parallel code grep.
  
```
Usage: ext [options] [name] ...
Options:
  -h [ --help ]                         Produce help message.
  -p [ --path ] arg (=/Users/dario/Documents/ext)
                                        The path to search in.
  -n [ --name ] arg                     The name to search for.
  -j [ --threads ] arg (=4)             Number of threads.
  -l [ --list ]                         List or search for files.
  -c [ --nocolor ]                      Disable color output.
  -q [ --quick ]                        Stop searching after first match.
```
